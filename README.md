# рџЌї WatchNext

**WatchNext** is a full-stack, AI-powered movie recommendation platform. It features a custom machine-learning backend built with FastAPI and a stunning, Netflix-inspired Flutter Web frontend. The platform seamlessly blends local recommendation models (using the MovieLens dataset) with the TMDB API to deliver high-quality posters, backdrops, and dynamic movie data.

## вњЁ Features

- **Personalized Recommendations:** Get tailored movie suggestions based on unique user profiles.
- **"More Like This":** Click on any movie to instantly view similar titles generated by the recommendation engine.
- **Cinematic UI:** A premium, dark-mode interface featuring dynamic hover animations, a hero banner, and horizontal scrolling rows (built with Flutter).
- **Rich Metadata:** Automatically fetches high-resolution posters, backdrops, and synopses from the **TMDB API**.
- **User Switching:** Instantly test different recommendation profiles by searching for different User IDs.

## рџљЂ Getting Started

Follow these steps to run WatchNext on your local machine.

### 1. Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (latest stable)
- [Python 3.9+](https://www.python.org/downloads/)
- A [TMDB API Key](https://developer.themoviedb.org/docs/getting-started)

### 2. Backend Setup
1. Navigate to the root directory.
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the FastAPI server locally:
   ```bash
   uvicorn api.main:app --reload
   ```
   *The API will be available at `http://127.0.0.1:8000`*

### 3. Frontend Setup
1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install Flutter packages:
   ```bash
   flutter pub get
   ```
3. Create a `.env` file in the `frontend` root directory and add your TMDB API Key:
   ```env
   TMDB_API_KEY=your_actual_tmdb_api_key_here
   ```
4. Run the Flutter web app:
   ```bash
   flutter run -d chrome
   ```
